Vayleryn

The Lord of the Craft

Website | BitBucket

The Lord of the Craft is a Minecraft roleplay server loosely based on The Lord of the Rings, I'm currently working on a dice combat plugin.



vychalo

GitHub Pages | GitHub

vychalo is a modular framework for Minecraft roleplay servers. It aims to revolutionise and unite servers and become a uniform mechanic in how they work, while being highly customisable and powerful.



BukkitPopulators

GitHub

BukkitPopulators (working title) is a project to bring together some of the best block populators for Bukkit in an easy-to-use, single install library. This has shown a fair amount of interest from various world generator authors.



ItemEnchanting

GitHub | BukkitDev | Curse

ItemEnchanting was my first released Bukkit plugin. It's a simple plugin that alters Minecraft enchanting mechanics to use items as opposed to experience.



Galaxy

GitHub | BukkitDev | Curse

Galaxy is a server GUI project for Bukkit, aiming to make the life of server admins easier and prettier!



Underdark
