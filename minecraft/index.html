Bukkit plugins

ItemEnchanting

ItemEnchanting allows you to use items rather than experience to enchant items.


KPerms

KPerms is a minimalistic permissions plugin that I've made several contributions to.


RealChop

RealChop allows trees to be affected by gravity and topple over. It was originally created by Vovics, but I now maintain it.


vychalo

vychalo is a pluggable roleplay framework.


Galaxy

Galaxy is a server GUI plugin.


Underdark

Underdark is a world generator loosely based on the underdark of Dungeons and Dragons.
